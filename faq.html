자주 묻는 질문

궁금한 점을 빠르게 찾아보세요.

  • Q 회원가입은 어떻게 하나요? ▼

    분양톡 홈페이지 상단 또는 우측 상단의 무료가입 버튼을 클릭하여 가입하실 수 있습니다.

    아이디(영문/숫자), 닉네임, 비밀번호를 입력하면 즉시 가입이 완료됩니다. 이메일 인증 없이 바로 이용 가능합니다.

  • Q 로그인이 안 됩니다. 어떻게 해야 하나요? ▼

    아이디와 비밀번호를 정확히 입력하셨는지 확인해 주세요. 분양톡은 아이디에 @bunyangtok.com이 자동으로 붙어 이메일 형식으로 저장됩니다.

    • 가입 시 입력한 아이디(예: love2455v)를 로그인 창에 입력하세요.
    • 비밀번호 확인 후에도 로그인이 안 되면 고객센터(📞 1668-3473)로 문의해 주세요.
  • Q 회원탈퇴는 어떻게 하나요? ▼

    현재 마이페이지 내 탈퇴 기능은 준비 중입니다. 탈퇴를 원하시면 고객센터(📞 1668-3473)로 문의해 주시면 즉시 처리해 드립니다.

    탈퇴 시 등록된 매물 및 개인정보는 관련 법령에 따라 일정 기간 보관 후 삭제됩니다.

  • Q 매물 등록은 유료인가요? ▼

    현재 무료로 운영 중입니다. 회원가입 후 누구나 무료로 매물을 등록하고 열람하실 수 있습니다.

    향후 유료 프리미엄 노출 기능이 추가될 수 있으며, 유료 전환 시 사전 공지 예정입니다.

  • Q 매물 등록 후 수정이나 삭제가 가능한가요? ▼

    네, 가능합니다. 로그인 후 내 현장 목록 페이지에서 직접 등록한 매물을 수정하거나 삭제할 수 있습니다.

    • 수정: 내 현장 목록 → 해당 매물 클릭 → 수정하기
    • 삭제: 내 현장 목록 → 해당 매물의 삭제 버튼 클릭
  • Q 등록한 매물이 검색에 바로 나오지 않아요. ▼

    등록 후 즉시 목록에 표시되는 것이 정상입니다. 만약 노출이 안 된다면 아래 사항을 확인해 주세요.

    • 브라우저 새로고침 후 다시 확인해 보세요.
    • 매물 내용이 운영 정책에 위반되는 경우 관리자 검토 중일 수 있습니다.
    • 그래도 문제가 지속되면 고객센터(📞 1668-3473)로 문의해 주세요.
  • Q 분양상담사로 취업하고 싶은데 어떻게 지원하나요? ▼

    구인 중인 현장을 찾으려면 구인구직매물 페이지 또는 지도매물 페이지에서 원하는 지역과 조건으로 검색하세요.

    관심 있는 현장의 상세 페이지에서 연락처(전화/카카오 오픈채팅)를 통해 직접 문의하시면 됩니다.

    • 경력·무경력 모두 지원 가능한 현장이 있습니다.
    • 분양상담사 경험이 없어도 교육 후 채용하는 현장도 있으니 각 현장 공고를 참고하세요.
  • Q 수수료 정보는 어떻게 확인하나요? ▼

    각 매물 상세 페이지에서 수수료율 또는 수수료 금액을 확인할 수 있습니다. 등록자가 공개한 범위 내에서 정보가 제공됩니다.

    구체적인 수수료 조건은 현장 담당자에게 직접 문의하시는 것이 가장 정확합니다.

  • Q 관심 현장을 저장하고 싶은데 어떻게 하나요? ▼

    매물 목록 또는 상세 페이지에서 하트(♡) 버튼을 누르면 관심 현장으로 저장됩니다.

    로그인 상태에서 저장한 관심 현장은 어느 기기에서도 동일하게 확인할 수 있습니다.

  • Q 서포터즈란 무엇인가요? ▼

    서포터즈는 분양 현장 정보를 분양톡에 공유하고 플랫폼 확산에 기여하는 자원봉사 + 혜택 프로그램입니다.

    • 현직 분양상담사 또는 분양 관련 종사자라면 누구나 신청 가능합니다.
    • 혜택: 매물 우선 노출, 서포터즈 뱃지, 소정의 활동 지원금

    자세한 내용은 서포터즈 페이지를 참고하세요.

  • Q 허위 매물을 발견했을 때 어떻게 신고하나요? ▼

    허위·과장 매물을 발견하셨다면 고객센터(📞 1668-3473)로 신고해 주세요. 확인 후 신속히 조치하겠습니다.

    신고 시 해당 매물 URL과 허위로 의심되는 내용을 함께 알려주시면 처리에 도움이 됩니다.
